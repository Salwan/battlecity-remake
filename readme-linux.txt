Needed dependencies:
- Python 2.5
- Pygame 1.9.1
- Psyco 1.6 (Optional)

To run the game on Linux, give execute right to "battlecity.py" (or battlecity_psyco.py) (chmod 0755) and execute it.
Or, from terminal write this:

/usr/bin/python battlecity.py

(If your python runtime is installed somewhere else, change the path)
